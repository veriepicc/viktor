## GitHub Integration Notes
- GitHub CLI: `coworker_github_cli(args=[...])` — async, needs `await`
- Response fields: `.stdout`, `.stderr`, `.exit_code`, `.success`
- Large stdout truncated with "Full output saved to: {path}" — must read file
- No `stdin_data` param — write JSON to temp file, use `--input <filepath>`
- Push via API PUT `/repos/{repo}/contents/{path}` with base64 content
- Must include `sha` field when updating existing files
