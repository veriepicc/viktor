# TinyTools

A collection of free, client-side web tools. Each tool runs entirely in the browser — no backend, no data collection, no tracking.

## Tools

| Tool | Description |
|------|-------------|
| [PixSqueeze](pixsqueeze/) | Image compressor — reduce file size in your browser |
| [JSONPrettify](jsonprettify/) | JSON formatter, validator & minifier |
| [HashCraft](hashcraft/) | MD5, SHA-1, SHA-256, SHA-512 hash generator |
| [PassCraft](passcraft/) | Strong password generator with strength meter |
| [ChromaForge](chromaforge/) | Color palette generator with export to CSS/Tailwind |
| [RegexLab](regexlab/) | Live regex tester with match highlighting |
| [CSSGrad](cssgrad/) | CSS gradient generator with live preview |
| [TextCount](textcount/) | Word counter, character counter & reading time |
| [BaseShift](baseshift/) | Number base converter (binary, hex, octal, etc) |
| [UnitFlip](unitflip/) | Universal unit converter |
| [DiffCheck](diffcheck/) | Text comparison & diff tool |
| [CronTranslate](crontranslate/) | Cron expression translator & generator |
| [LoremCraft](loremcraft/) | Placeholder text generator (multiple styles) |
| [MarkdownView](markdownview/) | Live Markdown editor with preview |
| [FaviconForge](faviconforge/) | Favicon generator from text, emoji or image |
| [MetaCheck](metacheck/) | Open Graph & meta tag previewer |
| [TableConvert](tableconvert/) | CSV to JSON/HTML/Markdown/SQL converter |
| [SVGMake](svgmake/) | SVG shape & pattern generator |
| [Blindfold](blindfold/) | AES-256 text encryption & decryption |
| [Timestamp](timestamp/) | Unix timestamp converter & live clock |

## Stack

- Pure HTML/CSS/JS — no frameworks, no dependencies
- 100% client-side — nothing is sent to any server
- Each tool is a single `index.html` file
- Designed for GitHub Pages hosting
